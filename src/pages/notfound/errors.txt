Код,Название,Краткое описание
400,Bad Request,Некорректный запрос (синтаксическая ошибка).
401,Unauthorized,Требуется аутентификация (логин/пароль).
402,Payment Required,Зарезервировано для будущих платежных систем.
403,Forbidden,Доступ запрещен (даже если вы авторизованы).
404,Not Found,Страница или файл не найдены.
405,Method Not Allowed,"Метод запроса (например, POST) не поддерживается для этого URL."
406,Not Acceptable,Сервер не может отдать контент в нужном формате (Encoding/Language).
407,Proxy Authentication Required,Нужна авторизация через прокси-сервер.
408,Request Timeout,Истекло время ожидания запроса от браузера.
409,Conflict,"Конфликт при выполнении запроса (например, при правке одного файла)."
410,Gone,Ресурс был удален навсегда (в отличие от 404).
411,Length Required,Не указана длина содержимого (Content-Length).
412,Precondition Failed,Одно из условий запроса не выполнено.
413,Payload Too Large,"Запрос слишком большой (например, файл весит больше лимита)."
414,URI Too Long,URL-адрес слишком длинный.
415,Unsupported Media Type,Неподдерживаемый тип данных (формат файла).
416,Range Not Satisfiable,Запрошен недопустимый диапазон байтов.
417,Expectation Failed,Сервер не может удовлетворить условиям заголовка Expect.
418,I'm a teapot,Первоапрельская шутка (протокол HTCPCP).
421,Misdirected Request,"Запрос направлен на сервер, который не может дать ответ."
422,Unprocessable Entity,"Ошибка в логике данных (например, неверный формат почты в форме)."
423,Locked,Ресурс заблокирован.
424,Failed Dependency,Ошибка из-за сбоя предыдущей операции.
426,Upgrade Required,"Сервер требует обновления протокола (например, на TLS 1.3)."
429,Too Many Requests,Слишком много запросов (защита от спама/DDOS).
431,Request Header Fields Too Large,Заголовки запроса слишком велики.
451,Unavailable For Legal Reasons,Доступ закрыт по юридическим причинам (цензура).

Код,Название,Краткое описание
500,Internal Server Error,"Общая ошибка сервера (самая частая, без деталей)."
501,Not Implemented,"Сервер не поддерживает метод, нужный для ответа."
502,Bad Gateway,Плохой шлюз (ошибка связи между двумя серверами).
503,Service Unavailable,Сервер перегружен или на тех-обслуживании.
504,Gateway Timeout,Истекло время ожидания ответа от вышестоящего сервера.
505,HTTP Version Not Supported,Версия протокола HTTP не поддерживается.
506,Variant Also Negotiates,Ошибка конфигурации (циклическая ссылка на ресурсы).
507,Insufficient Storage,На сервере закончилось место на диске.
508,Loop Detected,Сервер обнаружил бесконечный цикл при обработке запроса.
510,Not Extended,Нужны дополнительные расширения для выполнения запроса.
511,Network Authentication Required,"Требуется авторизация в самой сети (например, Wi-Fi в отеле)."

Код,Название,Контекст
444,No Response,Nginx закрывает соединение без ответа (защита от ботов).
499,Client Closed Request,"Клиент закрыл соединение до того, как Nginx успел ответить."
520,Unknown Error,"Cloudflare не понимает, что именно сломалось на сервере."
521,Web Server Is Down,Сервер полностью выключен или сбрасывает соединение.
522,Connection Timed Out,Cloudflare не смог подключиться к серверу за отведенное время.
524,A Timeout Occurred,"Соединение установлено, но сервер молчит слишком долго."
